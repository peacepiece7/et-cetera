# Flow chart 2

[flow chart reference](https://mermaid.js.org/syntax/flowchart.html)

빠르게 의미를 전달하고자 하는 이유로 사용한다면 subgraph나 스타일링을 사용하지 않는게 좋을듯 함

```mermaid
%% 플로우 차트 1
flowchart TD
 START([start])
 START --> A01{{page}} --> A01_IF{if logged in}
 A01_IF --> | yes | A02{{dashboard}}
 A01_IF --> | no | A03{{login}}
```
