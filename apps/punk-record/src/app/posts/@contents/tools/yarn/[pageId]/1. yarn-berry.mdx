# yarn

다음 명령어로 yarn 설치한다.

```bash
corepack enable
yarn set version stable
```

프로젝트 생성

```bash
yarn create next-app
yarn init -2
```

## vscode 설정

1. vscode extension에서 ZipFs를 설치

PnP 모드시 zip archives에서 타입을 가져와주는 yarn toolchain이다. [ZipFS](https://marketplace.visualstudio.com/items?itemName=arcanis.vscode-zipfs)  
(node_modules가 없기 때문에 타입 에러 발생한다. yarn pnp 모드시 필수)

2. `yarn dlx @yarnpkg vscode` 실행

간접 패키지를 설치해서 prettier, Typescript 등 패키지가 실행될 때 `require`를 호출하여 경로를 찾아주는 역할을 한다.

3. ctrl(command) + p => "> Typescript : Select Typescript version" 옵션 선택 => .yarn/sdks/typescript... 선택

만약 "> Typescript : Select Typescript version"가 나오지 않는다면 .ts 파일로 이동해서 다시 해본다.
