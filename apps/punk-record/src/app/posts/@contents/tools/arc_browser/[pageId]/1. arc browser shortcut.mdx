# arc browser 주요 사용법 정리

## 용어정리

### Space

작업 공간(workspace) 말한다.

예를들어 vscode 실행하고 intellij도 실행하면 Space가 두 개인데 arc browser는 이걸 Space라고 부른다. (`ctrl + alt + l/r arrow`로 스페이스 전환)

### Pinned Page

고정된 페이지들이다.

예를들어 vscode에서 작업중인 탭을 여러개 띄우고 `ctrl + 1, 2` 또는 `ctrl + tab`으로 이동하는데 이럴떄 쓴다.

<MDXImage src="/images/arc_5.png" width={300} className="m-2" />

### Pinned Tab

Pinned Page아래에 있는 탭이다.

Pinned Page랑 마찬가지이고 폴더로 묶어서 관리할 수 있다.

`ctrl + t`를 누르면 Pinned Tab에 있는 페이지를 검색할 수 있다.

<MDXImage src="/images/arc_6.png" width={300} className="m-2" />

## 주요 shortcut

앵간한건 Chrome이랑 같다.

### 탭 추가

`ctrl + t`

### 검색

검색할 때 히스토리가 선택되거나 탭이 선택되는 경우가 있는데 **스페이스바 한 번 눌러**주면된다.

`ctrl + l`

### 스페이스 전환

`ctrl + alt + left/right arrow`

### 탭 전환

`ctrl + tab`

사이드바 아이콘의 색상이 진해지면 탭을 전환 할 때 대화상자에 포함되기에

창이 너무 많으면 미리미리 꺼두자.

조오금 아쉬운게 탭 전환 대화상자에서 방향키로 이동이 안된다 ㅜ

### 탭 분리, 취소

탭 분리 : `ctrl + shift + \+`

취소 : `ctrl + shift + \-`

드래그로 탭 분리도 됨

### Favorite, Pinned 탭 이동

`ctrl + number`

Favorite -> Pinned Tab 순서로 이동됨

### 사이드 바 열기 닫기

`ctrl + s`

### 링크 복사

`ctrl + shift + c`

### Tab \<=> PinnedTab 이동

`ctrl + d`

### 다운로드 목록, 설정, 플레그 확인

다운로드해도 반응이 없는 경우가 있는데 다운로드 목록에서 확인할 수 있다.

`ctrl + l` 입력 후 `arc://downloads`를 입력한다. (arc 아이콘이 있는 목록을 클릭, 아무거나 선택하면 검색창에 `arc://downloads`가 입력된다.)

<MDXImage src="/images/arc_search.png" width={450} />

플래그는 `arc://flags`, 설정은 `arc://settings`로 확인할 수 있다.
