# 공변성

좁은 타입을 넓은 타입에 대입 할 수 있다.

ts에서는 리턴 값에 대해서 공변성을 가진다.

`type One = (i : number) => number | string`일 경우
리턴 타입이 `number`, `string` ,`number | string`은 되지만

`number | string | boolean`, `boolean`은 안된다.

# 반공변성

넓은 다음에 좁은 타입을 대입 할 수 있다.

`type Two = (i : string | number) => number`일 경우

매개변수 i는 `string | number`, `string | number | boolean`는 되지만

`string`, `number`는 안된다.

# 이변성

공변성과 반공변성을 모두 가질 수 있다.

```ts
interface Object {
  method1(p: number | string | boolean): number
  method2: (p: string | number | boolean) => number
}
```

이런 경우 method2는 반공변성을 가지지만 method1은 이변성을 가지기 때문에, 매개변수 p가 `number`, `string` 등 다양한 타입을 가질 수 있다.
