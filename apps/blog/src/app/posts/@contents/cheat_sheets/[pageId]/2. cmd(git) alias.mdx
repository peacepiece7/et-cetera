# CMD

## 자동 실행

cmd 바로가기 => 속성 => 대상(T)

```
%windir%/system32/cmd.exe \k %userprofile%/cmd_alias.exe
```

win + R => regdit 이동

HEY_LOCAL_MACHINE\\SOFTWARE\Microsofte\\Command Processor 들어가서

문자열 값 => AutoRun 또는 autorun 둘 중 하나 => 값 데이터에 `%userprofile%/cmd_alias.exe` 추가

## cmd alias

```
@echo off
REM # Compatibility
doskey ls=dir
doskey pwd=cd ,
doskey clear=cls

REM # Push cycle
doskey ga=git add $1
doskey gaa=git add .
doskey gc=git commit $1 $2
doskey gph=git push $1 $2

REM # Pull
doskey gfa=git fetch --all
doskey gp=git pull $1 $2

REM # Health check
doskey gl=git log
doskey gs=git status
doskey glo=git log --oneline
doskey glg=git log --graph
doskey glog=git log --oneline --graph

REM # Branch
doskey gb=git branch $1 $2
doskey gco=git checkout $1 $2
doskey gsw=git switch $1 $2
doskey gpick=git cherry-pick $1 $2

REM # Stage
doskey grt=git restore $1 $2
doskey grts=git restore --staged $1

REM # NPM
doskey nr=npm run $1
doskey nrd=npm run dev
doskey nrs=npm run serve
doskey nrb=npm run build

REM # PNPM
doskey pn=pnpm $1 $2 $3
doskey pnd=pnpm run dev
doskey pnrs=npm run serve
doskey pnrb=npm run build

REM # Path
doskey repo=cd %userprofile%\repo\peacepiece7
doskey obs=%userprofile%\AppData\Local\Programs\obsidian\Obsidian.exe
doskey cobs=cd %userprofile%\repo\peacepiece7\obs-repo
doskey p=cd %userprofile%\repo\projects
doskey ps=cd %userprofile%\repo\projects\sports-club
doskey psa=cd %userprofile%\repo\projects\sports-club\assets
@echo on
```
